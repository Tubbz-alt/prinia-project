README

How to access the code:

Requirements
1. MATLAB 2012b or later

Steps
1. Download the folder prinia_project_git to a folder
2. In MATLAB change current folder to prinia_project_git

A. Open signal_space_partitioning.mlx from MATLAB window for signal space analysis comparing different species
B. Open note_space_analysis.mlx from MATLAB window for characterizing the song of different species
C. Open repetition_rate_analysis.mlx from MATLAB window for characterizing the repetition rate of the notes (in the case of birds which sing only 1 note)
